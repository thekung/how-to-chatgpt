# Module 1 : เข้าใจ AI และ Generative AI

# AI คืออะไร

AI (Artificial Intelligence)

คือ

> "เทคโนโลยีที่ทำให้คอมพิวเตอร์สามารถเรียนรู้ วิเคราะห์ และตัดสินใจในลักษณะที่คล้ายมนุษย์"

AI ไม่ใช่โปรแกรมธรรมดาที่เขียนคำสั่งตายตัวแต่สามารถ

- เรียนรู้ข้อมูล
- วิเคราะห์ข้อมูล
- คาดการณ์ผลลัพธ์
- สร้างคำตอบใหม่ได้

---

## ตัวอย่าง AI รอบตัวเรา

| การใช้งาน | AI ทำอะไร |
|------------|-----------|
| Google Maps | คำนวณเส้นทาง |
| Facebook | เลือกโพสต์ที่ควรเห็น |
| Netflix | แนะนำหนัง |
| Spotify | แนะนำเพลง |
| Shopee | แนะนำสินค้า |
| Gmail | ตรวจจับ Spam |
| Siri | ผู้ช่วยเสียง |
| ChatGPT | ผู้ช่วยตอบคำถามและสร้างเนื้อหา |

---

AI ไม่ได้มีแค่ ChatGPT

```
Artificial Intelligence
│
├── Machine Learning
│
├── Deep Learning
│
├── Computer Vision
│
├── Speech Recognition
│
└── Generative AI
```

## Machine Learning

Machine Learning

คือ AI ที่เรียนรู้จากข้อมูล

ตัวอย่าง

- ทำนายราคาบ้าน
- ตรวจจับการโกงบัตรเครดิต
- วิเคราะห์ยอดขาย

---

## Computer Vision

AI ที่เข้าใจภาพ

ตัวอย่าง

- Face ID
- ตรวจจับใบหน้า
- อ่านทะเบียนรถ
- ตรวจสอบสินค้าเสียหาย

---

## Speech Recognition

AI ที่เข้าใจเสียง

ตัวอย่าง

- Siri
- Google Assistant
- Alexa
- Voice Search

---

# Generative AI คืออะไร

Generative AI

คือ AI ที่สามารถ "สร้าง" เนื้อหาใหม่ได้

เช่น

- เขียนข้อความ
- สร้างรูปภาพ
- แต่งเพลง
- เขียนโปรแกรม
- สร้างวีดีโอ
- สร้าง Presentation

ต่างจาก AI แบบเดิมที่เน้นการวิเคราะห์

# AI vs Search Engine

หลายคนเข้าใจผิดว่า ChatGPT คือ Google

จริง ๆ แล้วต่างกันมาก

| Search Engine | AI |
|---------------|----|
| ค้นหาเว็บไซต์ | สร้างคำตอบ |
| แสดงลิงก์ | อธิบายให้ |
| ต้องเปิดหลายเว็บ | สรุปให้ทันที |
| หาข้อมูล | วิเคราะห์ข้อมูล |
| ไม่ช่วยเขียน | ช่วยเขียนได้ |

# ข้อดีของ AI

- ทำงานเร็ว
- ประหยัดเวลา
- สรุปข้อมูลจำนวนมาก
- ช่วยคิดไอเดีย
- ทำงานได้ตลอดเวลา
- รองรับหลายภาษา

---

# ข้อจำกัดของ AI

AI ไม่ได้ถูกต้อง 100%

อาจเกิด

- Hallucination
- ข้อมูลล้าสมัย
- ตอบผิดบริบท
- ตอบมั่นใจแต่ผิด

ดังนั้น

**ต้องตรวจสอบข้อมูลทุกครั้ง**

---

# Golden Rules

ก่อนนำ AI ไปใช้งานจริง

ควรจำไว้เสมอ

✅ AI ช่วยคิด ไม่ใช่คิดแทน

✅ ตรวจสอบข้อมูลสำคัญเสมอ

✅ อย่าใส่ข้อมูลลับ

เช่น

- Password
- ข้อมูลลูกค้า
- เงินเดือน
- เลขบัตรประชาชน

# LAB 1 : AI กับ Search Engine

## วัตถุประสงค์

เปรียบเทียบการใช้งาน Google กับ ChatGPT

---

### ขั้นตอน

ค้นหาใน Google

```
Linux KVM Installation
```

สังเกต

- ได้เว็บไซต์กี่เว็บ
- ต้องอ่านเองหรือไม่

---

จากนั้นถาม ChatGPT

```
ช่วยสอนติดตั้ง KVM บน Ubuntu Server แบบละเอียด
```

เปรียบเทียบ

- ความเร็ว
- ความเข้าใจ
- จำนวนขั้นตอน
- ความครบถ้วน

---

## สิ่งที่ได้เรียนรู้

- Google เหมาะกับค้นหา
- ChatGPT เหมาะกับอธิบาย

---

# LAB 2 : ให้ AI อธิบายเรื่องยาก

Prompt

```
อธิบาย Kubernetes ให้เหมือนสอนนักเรียนมัธยม
```

จากนั้นลองเปลี่ยนเป็น

```
อธิบาย Kubernetes ให้เหมือนสอนผู้บริหาร
```

และ

```
อธิบาย Kubernetes สำหรับ System Administrator
```

สังเกต

AI จะเปลี่ยนระดับภาษาเอง
