# Module 2 : เข้าใจข้อดี ข้อจำกัดของ ChatGPT

# ChatGPT คืออะไร

ChatGPT คือ Large Language Model (LLM)

ถูกพัฒนาขึ้นเพื่อ

- ตอบคำถาม
- อธิบายข้อมูล
- วิเคราะห์ข้อความ
- เขียนเอกสาร
- เขียนโปรแกรม
- สรุปรายงาน
- แปลภาษา
- ช่วยคิดไอเดีย

ChatGPT ไม่ใช่ฐานข้อมูล

แต่เป็นโมเดลที่เรียนรู้รูปแบบของภาษาและความสัมพันธ์ของข้อมูลจำนวนมหาศาล

# ข้อจำกัดของ ChatGPT

แม้ ChatGPT จะมีความสามารถสูง แต่ยังมีข้อจำกัดหลายประการ เช่น

- อาจตอบผิด
- Hallucination - สร้างข้อมูลขึ้นมาเองทั้งที่ข้อมูลนั้นไม่มีจริง เช่น
  - อ้างอิงงานวิจัยปลอม
  - สร้างชื่อหนังสือ
  - สร้าง URL
  - สร้างชื่อบุคคล
  - สร้างเลขหน้าเอกสาร
- ไม่รู้ทุกเรื่อง เช่น
  - ข้อมูลภายในบริษัท
  - เอกสารลับ
  - ข้อมูลล่าสุดที่เพิ่งเกิดขึ้น
  - ระบบเฉพาะองค์กร
- ขึ้นอยู่กับ Prompt - Prompt ที่ไม่ดี มักได้คำตอบไม่ดี
- ไม่เข้าใจบริบททั้งหมด ไม่สามารถอ่านใจผู้ใช้
- ความรู้เฉพาะทาง เช่น
  - กฎหมาย
  - การแพทย์
  - ภาษี
  - การลงทุน
- ไม่มีประสบการณ์จริง AI สามารถอธิบายได้แต่ไม่ได้ทดลองจริง
ดังนั้นผลลัพธ์บางอย่างต้องนำไปทดลองเอง

---

# การตรวจสอบความถูกต้องของข้อมูล

ทุกครั้งก่อนนำ AI ไปใช้งานจริงควรตรวจสอบข้อมูลและความถูกต้อง แหล่งอ้างอิงจริงเพื่อยืนยันผลลัพธ์เสมอ ง่ายที่สุดคือการ "ถามซ้ำ"

---

# เทคนิคการใช้ ChatGPT อย่างปลอดภัย

## อย่าใส่ข้อมูลลับ

ตัวอย่าง

- Password
- API Key
- Source Code ภายใน
- ข้อมูลลูกค้า
- ข้อมูลเงินเดือน
- ข้อมูลสุขภาพ
- ข้อมูลส่วนบุคคล
- ใช้ AI เป็นผู้ช่วยไม่ใช่ผู้ตัดสินใจแทนทั้งหมด
