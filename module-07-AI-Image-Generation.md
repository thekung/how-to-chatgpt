# Module 7 : ใช้ AI สร้างรูปภาพ (AI Image Generation)

# AI Image Generation คืออะไร

AI Image Generation คือ

> การใช้ปัญญาประดิษฐ์สร้างภาพจากข้อความ (Text Prompt)

ผู้ใช้เพียงอธิบายภาพที่ต้องการ AI จะสร้างภาพใหม่ขึ้นมา โดยไม่ต้องวาดเอง

---

# ตัวอย่างงานที่ AI สร้างได้

- Presentation
- Poster
- Infographic
- Illustration
- Character
- Icon
- Product Mockup
- Architecture Diagram
- Background
- Banner
- Social Media

---

# Prompt คือหัวใจสำคัญ

AI จะสร้างภาพจากข้อความ ยิ่ง Prompt ชัดเจน ภาพยิ่งตรงความต้องการ

---

# โครงสร้าง Prompt

Prompt ที่ดีควรประกอบด้วย

- Subject
- Style
- Composition
- Lighting
- Color
- Aspect Ratio
- Quality


---

# Subject
